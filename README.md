# 🤖 AI Chatbot Web Application

An AI-powered chatbot web application that enables real-time conversations using modern Natural Language Processing (NLP) techniques. The project features a clean, responsive UI inspired by ChatGPT and supports intelligent responses, chat history, and additional utilities like summarization and sentiment analysis.

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/kamleshkumarr/CHATBOT.git
cd chatbot-project
```

2. Create a virtual environment:

```bash
python -m venv venv
venv\Scripts\activate   # For Windows
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Add your API key in `.env` file:

```
API_KEY=your_api_key_here
```

5. Run the application:

```bash
python app.py
```

6. Open in browser:

```
http://127.0.0.1:5000
```
